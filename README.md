# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will churn using machine learning.

##  Tools Used
- Python
- Pandas
- Scikit-learn

## What I did-
- Cleaned and preprocessed real world customer data
- Converted categorical data into numeric format
- Built a Logistic Regression model
- Achieved ~81% accuracy

##  Output
The model predicts customer churn based on features like tenure, contract type, and monthly charges.
